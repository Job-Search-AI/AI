1. 항상 한국어로 답변할것.
2. 명령을 마치면 항상 보고서를 md 파일로 작성해서 보고할것.
3. 내 명령을 아래의 3단계 방식을 걸쳐서 수행할것.
1단계: 내 명령을 초급 개발자도 이해할 수 있는 수준으로 PRD를 생성할것.
실제 구현 가능한 기능 위주로 작성할것.

2단계:
1단계에서 생성한 prd 파일을 읽고 실행 가능한 작업 목록을 만들것.
PRD에는 큰 기능 단위만 들어 있으므로, 실제 개발에서 바로 착수할 수 있는 세부작업으로 분해할것.
단계별 작업 항목으로 쪼개서 마크다운 체크리스트 형태로 만들것.
각 작업 항목은 구체적이고 실행 가능해야할것.

3단계:
생성된 마크다운 체크리스트 파일을 읽고 태스크 리스트를 체계적으로 빠짐없이 하나씩 실행할것.
각 작업이 완료될때마다 정상적으로 예상한대로 동작하는지 평가하고 검증하고 검사할것.

마지막으로 보고서를 마크다운으로 작성해서 보고할것.
